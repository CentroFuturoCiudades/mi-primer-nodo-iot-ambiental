Sesión 2 — Tecnología para medir el entorno

Fecha: 20 de junio de 2026
Horario: 9:00 am - 1:00 pm

Propósito​

Construir la base técnica del nodo IoT: microcontrolador, programación, sensores, visualización local y comunicación.

Cada práctica sigue una secuencia común: fundamento técnico, armado del sistema y programación guiada. La codificación por color permite reconocer con rapidez el tipo de trabajo de cada bloque.

A1

Mi primer programa. Control de un LED como salida observable del microcontrolador.

A2

Lectura ambiental. Obtención de mediciones del sensor mediante I2C.

C1

Comunicación del nodo. Envío de mediciones mediante Notecard y LoRa.
