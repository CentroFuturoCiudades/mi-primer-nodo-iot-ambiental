# Ayuda

## No aparecen mensajes en la computadora

El panel de salida de Arduino IDE muestra principalmente información de
compilación. Para observar los mensajes enviados por el programa, abre el
monitor serial y verifica que la velocidad configurada coincida con la indicada
en el código.

## Error al cargar el programa

Revisa que la tarjeta esté conectada mediante un cable USB de datos y que se
encuentre en modo de arranque cuando el método de carga lo requiera. Si la carga
falla, repite la secuencia **BOOT + RESET** y vuelve a intentarlo.

## Falta una biblioteca

Instala la biblioteca desde el administrador de bibliotecas de Arduino IDE.
Después de instalarla, reinicia Arduino IDE si el entorno no la reconoce de
inmediato.
