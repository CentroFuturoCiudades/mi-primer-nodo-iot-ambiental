# Guía general del taller

Este sitio reúne los materiales académicos del taller **Ciencia, Tecnología y
Arte para el cuidado del Arroyo Vivo**. Su propósito es ofrecer una guía de
consulta ordenada para revisar presentaciones, prácticas de laboratorio,
bitácoras y productos de cada sesión.

## Ruta de aprendizaje

El taller avanza de manera gradual:

1. Comprender el problema ambiental y formular preguntas de observación.
2. Estudiar los fundamentos del microcontrolador y su relación con sensores.
3. Registrar mediciones ambientales y revisar su calidad.
4. Interpretar datos para construir propuestas sustentadas en evidencia.

## Uso recomendado

Consulta primero la sección **Taller completo** para ubicar la sesión
correspondiente. Después revisa las actividades técnicas en el orden indicado,
ya que cada una introduce conceptos que serán utilizados en la siguiente.
